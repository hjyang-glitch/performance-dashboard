# 퍼포먼스 대시보드

퍼포먼스 마케팅팀 일일 확인용 Streamlit 대시보드.

## 로컬 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

별도 인증 설정 없이 바로 실행됩니다.

## 데이터 소스

`data.py`에 임시 샘플 데이터가 하드코딩되어 있습니다. 채널+앱스플라이어 조인 파이프라인이
완성되면 `load_campaign_data()` 내부만 실제 데이터 소스(시트/BigQuery 등) 조회로 교체하면 됩니다.

## 현재 범위

- 🔴 오늘의 알림 탭만 실제 구현됨 (전일 대비 CPA ±30% 이상 급변 시 알림, 원인 소재 CTR 하락 함께 표시)
- 채널 성과 / 소재 성과 / AB테스트 / 예산 페이싱 탭은 후속 플랜에서 구현 예정
